SenseLab is the merge-to-incident lineage layer your SDLC is missing. It tracks every line of AI-generated code from the moment it's written to the moment it ships — and ties production incidents back to the exact PR, model, and developer that introduced them.

## Get started in minutes

No repository connection required. Install the CLI, sign up, and start tracking AI lineage immediately.

```bash
# Install the SenseLab CLI
curl -fsSL https://cli.sense-lab.ai/install.sh | sh

# Sign up and authenticate
sense-cli signup

# Enable AI lineage tracking in your repo
sense-cli lineage enable
```
