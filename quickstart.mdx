## Get started in 3 steps

<Steps>
  <Step title="Install the SenseLab CLI">
    Run the following command in your terminal to install the CLI:

    ```bash
    curl -fsSL https://cli.sense-lab.ai/install.sh | sh
    ```
  </Step>

  <Step title="Sign up">
    Create your account directly from the CLI. This will open the SenseLab web UI in your browser to complete sign up with GitHub or Google.

    ```bash
    sense-cli signup
    ```

    <CardGroup cols={2}>
      <Card title="GitHub" icon="github" href="https://app.raia.live">
        Sign up with GitHub
      </Card>
      <Card title="Google" icon="google" href="https://app.raia.live">
        Sign up with Google
      </Card>
    </CardGroup>

    Once sign up is complete, your CLI is automatically authenticated — no extra steps needed.
  </Step>

  <Step title="Enable AI Lineage">
    Navigate to any local directory where your code lives (cloned from git) and run:

    ```bash
    sense-cli lineage enable
    ```

    That's it. SenseLab will start sending telemetry to your dashboard immediately.
  </Step>
</Steps>
